## SkillSwap – A Local Skill Exchange Platform

A responsive skill-sharing web platform where users can offer, learn, and exchange skills within their local community. Users can browse skill listings, view details, authenticate using Firebase, and manage their profiles.

### Live URL

- Live Website: https://mellow-otter-666845.netlify.app/

 ## Project Purpose

The purpose of SkillSwap is to connect learners with local skill providers. Whether it’s guitar lessons, language practice, coding help, or yoga sessions, SkillSwap helps users find the right skill provider and book a session easily.

## Tech Stack Used

### Frontend

- React

- React Router

- Firebase Authentication

- Tailwind CSS / DaisyUI

- Swiper.js

- react-hot-toast

 ## Main Features

- Responsive Navbar with Login/Signup and Profile options

- Firebase Authentication (Email/Password + Google)

- Protected Skill Details Page

- Skill Listings loaded from local JSON

- Booking form with success toast

- My Profile page with user information

- Update Profile option using updateProfile()

- Hero slider using Swiper.js

- Popular Skills Section

- Additional Homepage Section

- Persistent SPA routing with no reload errors

- Environment variables for Firebase config

- Google Password Reset (Forgot Password) feature

- Password visibility toggle in Login and Signup

## Overview

 SkillSwap is a modern web application that enables users to offer, learn, and exchange skills locally — from music lessons and language practice to coding help or fitness training. The platform features secure Firebase authentication, detailed skill listings, protected booking forms, profile management, and a responsive, minimalist design for a seamless user experience.
